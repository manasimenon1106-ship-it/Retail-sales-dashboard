# Retail Sales Dashboard

## Overview

The **Retail Sales Dashboard** is an interactive data visualization project built using **Python, Streamlit, Pandas, and Plotly**.
It helps analyze retail sales performance by showing important business metrics, trends, and insights in a simple dashboard format.

This dashboard is useful for understanding:

* Total sales performance
* Product/category-wise analysis
* Regional sales trends
* Profit and revenue insights
* Customer and order patterns

---

## Features

* Interactive dashboard built with **Streamlit**
* Data cleaning and analysis using **Pandas**
* Attractive visualizations using **Plotly**
* Business insights from retail sales data
* Easy-to-use filters for better analysis

---

## Tech Stack

* **Python**
* **Streamlit**
* **Pandas**
* **Plotly Express**

## Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/manasimenon1106-ship-it/Retail-sales-dashboard.git
cd Retail-sales-dashboard
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Streamlit app

```bash
streamlit run app.py
```
